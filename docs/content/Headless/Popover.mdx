<Header title="Popover" section=" ">
  Popovers are floating panels with arbitrary content.
</Header>

Typical use cases for popovers include user-interactive elements like action menus,
custom notifications, form element suggestions, content pickers, or help dialogs.

Popovers can also be used for big hideable sidebars, like a shopping cart or action panels.
