## How to use

1. Install the `jinjax-ui` python library doing

  ```bash
  pip install jinjax-ui
  ```

  **-- or --**

  Copy the components you need [from Github]({{metadata.repo}}) to a folder into your project.



2. Add the folder to your *JinjaX* catalog:

  ```python
  import jinjax_ui

  catalog.add_folder(jinjax_ui.components_path, prefix="ui")
  ```

3. Insert the `<ui:Globals />` component at the beginning of your base template, to load the shared UI assets:

  ```html+jinja
  <ui:Globals/>

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="utf-8">
      {{ catalog.render_assets() }}
      ...
  ```

4. Use the UI components in your components/templates:

  ```html+jinja
  <ui:Alert> ... </ui:Alert>
  ```
