## How to use

1. Install the `jinjax-ui` python library doing

  ```bash
  pip install jinjax-ui
  ```

2. Add it to your *JinjaX* catalog:

  ```python
  import jinjax_ui

  catalog.add_folder(jinjax_ui.components_path, prefix="ui")
  ```

3. Insert the `<ui:JXUI/>` component at the beginning of your base template, to load the shared UI assets:

  ```html+jinja
  <ui:JXUI/>

  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="utf-8">
      {{ catalog.render_assets() }}
      ...
  ```

4. Use any of the UI components in your own components/templates:

  ```html+jinja
  <ui:Alert> ... </ui:Alert>
  ```
