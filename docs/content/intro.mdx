## How to use

1. Install the `jinjax-ui` python library doing

  ```bash
  pip install jinjax-ui
  ```

2. Add it to your *JinjaX* catalog:

  ```python
  import jinjax_ui

  catalog.add_folder(jinjax_ui.components_path)
  ```

3. Use the UI components in your components/templates:

  ```html+jinja
  <ui:Alert> ... </ui:Alert>
  ```
