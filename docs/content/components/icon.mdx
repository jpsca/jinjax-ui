The `<Icon>` component is a simple wrapper around an `<i>` element styled using the [Material Icons](https://fonts.google.com/icons) font.

The other components in the library use this component to display icons, so you can override it to customize the icon styles.

## Usage

<Example name="ex.Icon"/>

The size and color of the icon is determined by the font size of the parent element


## Component arguments

### `Icon`

<Arguments>
  name | str  | **Required**. Name of the [Material Symbol](https://fonts.google.com/icons) icon to show.
  fill | bool | Whether to fill the icon with color. Default is `false`.
</Arguments>
